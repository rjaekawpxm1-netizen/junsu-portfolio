— About

공공데이터와 웹 데이터를 활용해 “어떤 문제를 해결할 것인지”를 먼저 정의합니다.

그에 맞는 데이터 수집·전처리·분석 과정을 설계하고, 실제 사용자에게 도움이 되는 결과를 만드는 것을 목표로 합니다.

단순 리포트가 아닌, 웹 서비스/대시보드 형태로 구현하는 작업을 좋아합니다.
